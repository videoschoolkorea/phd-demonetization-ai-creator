## 🔬 연구 개요

### 연구 목적
유튜브 알고리즘 기반 Demonetization 정책이 AI 생성 콘텐츠 크리에이터의 경제적 생존과 콘텐츠 전략에 미치는 영향을 실증적으로 규명

### 연구 방법 (혼합 연구)
- **알고리즘 감사**: YouTube Data API v3 활용
- **심층 인터뷰**: AI 크리에이터 30명 + 전통 크리에이터 20명
- **정책 문서 분석**: 유튜브 정책 변화 추적
- **비교 사례 연구**: 한국/미국/유럽 비교

### 이론적 배경
- Platform Capitalism (플랫폼 자본주의)
- Algorithmic Governance (알고리즘 거버넌스)
- Creator Economy & Digital Labor
