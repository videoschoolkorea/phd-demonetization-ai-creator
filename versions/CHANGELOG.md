## v1.3 — 2026-03-15 (커밋: ad00401)
**변경 유형**: 내용 보강 (연구 필요성 + 선행연구 + 문헌DB)

### 변경 섹션
- **연구의 필요성**: 한국 실제 수익 정지 사례 데이터 추가
- **선행연구 검토**: 국내 언론 보도 섹션 신설 (2건)

### 주요 변경 내용
1. **연구의 필요성 강화**
   - 한국이 AI Slop 채널 세계 조회수 1위(84억5,000만 회) 데이터 추가 (카프윙 분석, 머니투데이 2026.01.16 인용)
   - 한국 대표 사례 '3분 지혜' 채널 20억2,000만 회 후 수익 정지 사례 명시
   - 2026년 1~2월 X(트위터)·크리에이터 커뮤니티 내 수익 정지 경험 공유 급증 현황 추가
   - 매일경제(2026.01.19) 보도 인용

2. **선행연구 검토 — 국내 언론 보도 섹션 신설**
   - 허서윤 (매일경제, 2026.01.19): AI 쇼츠 수익 정지 철퇴 보도
   - 김지영 (머니투데이, 2026.01.16): 한국 AI Slop 세계 1위 보도

3. **문헌 데이터베이스**: 12편 → 14편 (국내 5편 → 7편)

### 변경 이유
- 연구 대상이 한국 사례임에도 실제 한국 사례 근거가 부족하다는 검토
- 학문적 필요성 입증을 위한 실증 데이터 보강

## v1.2 — 2026-03-15 (커밋: 7879b60)
**변경 유형**: 연구 대상 수정 (한국 중심 재설계)

### 변경 섹션
- **연구 대상 ③ 온라인 커뮤니티**: Reddit(미국) → 한국 커뮤니티로 전면 교체
- **연구 방법 ② 넷노그라피**: 수집처 한국 중심 재기술
- **연구 절차 1·3단계**: 넷노그라피 수집처 교체

### 주요 변경 내용
| 항목 | 변경 전 | 변경 후 |
|------|---------|---------|
| 주요 커뮤니티 | Reddit r/PartneredYoutube | **네이버 카페** (유튜버 관련 대형 카페) |
| 수집 방법 | PullPush API | **네이버 검색 API** |
| 보조 커뮤니티 | Reddit r/NewTubers | **디시인사이드 유튜버 갤러리 / 에펨코리아** |
| 영상 댓글 | 미포함 | **YouTube Data API v3 commentThreads** (한국어) |
| 비공개 채널 | 미포함 | **카카오톡 오픈채팅** (동의 취득 후 참여 관찰) |
| Reddit 위치 | 주요 데이터 | **한-미-유럽 비교용 보조 데이터**로 조정 |

### 변경 이유
- 연구 제목이 "한국 사례를 중심으로"임에도 데이터 수집처가 미국 Reddit 중심이었던 문제 해결
- 한국 연구의 타당성(validity) 확보

## v1.1 — 2026-03-15 (커밋: fd2421d)
**변경 유형**: 주요 섹션 대폭 보강 (연구 대상·방법·절차)

### 변경 섹션
- **연구 대상**: 단순 1줄 → 4개 층위 세분화
- **연구 방법**: 단순 나열 → 삼각검증(Triangulation) 4방향 체계화
- **연구 절차**: 개략적 4단계 → 세부 일정 및 완료 현황 포함

### 주요 변경 내용

**연구 대상 (4개 층위)**
- ① 한국 크리에이터 인터뷰 (AI 30명 + 전통 20명, 목적적 표집 기준 명시)
- ② 플랫폼 공개 데이터 (YouTube Data API v3, Social Blade/Apify, 학술 데이터셋 2차 분석)
  - Dunna et al.(2022) 354,000개 영상 데이터셋 저자 공유 요청
  - Kaggle YouTube Tech Channels 2025 공개 데이터셋
- ③ 온라인 커뮤니티 (Reddit PullPush API, Wayback Machine)
- ④ 정책 문서 (2025년 7월 기준점 명시, 방통위·KISDI·KMCNA)

**연구 방법 (삼각검증 4방향)**
1. 알고리즘 감사 (YouTube Data API v3, Social Blade Apify)
2. 넷노그라피 (Reddit PullPush API)
3. 심층 인터뷰 (50명, 반구조화)
4. 정책 문서 분석 + 비교 사례 연구

**연구 절차 (4단계 세부화)**
- 1단계(2026.1~3): 파일럿 데이터 수집 완료 내용 반영
- 2단계(2026.4~6): 정량 본수집
- 3단계(2026.7~9): 정성 수집
- 4단계(2026.10~2027.6): 통합 분석·논문 작성

### 변경 이유
- 실제 데이터 수집 스크립트 작성 및 파일럿 수집 완료 후 방법론 현실화
- 학술 데이터셋 2차 분석 전략 추가
